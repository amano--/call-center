# Project Context (GEMINI.md)

このファイルは、プロジェクトのコンテキストやルールを AI（Antigravity）と共有するためのものです。

## Skills

### UI Skills

UI やフロントエンドの実装を行う際は、以下のスキル定義ファイルに記載されたルール（制約、ベストプラクティス）に **厳密に従ってください**。

- **定義ファイル**: `./antigravity/skills/ui-skills/SKILL.md`

**主なルール概要:**

- Tailwind CSS のデフォルトを優先する
- アニメーションには `motion/react` を使用する
- アクセシビリティ（`aria-label` 等）を必ず考慮する
- インタラクションのフィードバックは 200ms 以内に収める
- 色やスペーシングは既存のトークン・変数を再利用する

## コマンド

- `/ui-skills`: 現在のコードがスキル定義に準拠しているかレビューを行います。
- `/biz-growth`: 現在のコンテンツや戦略が成長/販売戦略のスキル定義に準拠しているかレビューを行います。

### Business Growth Skills

ビジネス・マーケティング・コンテンツ制作関連のタスクを行う際は、以下のルールに従ってください。

- **定義ファイル**: `./antigravity/skills/biz-growth/SKILL.md`

**主なルール概要:**

- 結論ファースト、根拠、行動提案の順で構成する
- 数値は一次情報を優先し、比較表を作成する
- 商用クオリティ（High Detail, Cinematic 等）を前提とする
