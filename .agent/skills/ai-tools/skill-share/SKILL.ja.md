# スキル共有

## このスキルを使用するタイミング

以下が必要な場合にこのスキルを使用します：

- **新しいClaudeスキルを作成** 適切な構造とメタデータを備えた
- **スキルパッケージを生成** 配布の準備ができた
- **作成されたスキルを自動的にSlackで共有** チーム可視性のために
- **配布前にスキル構造を検証** する
- **スキルをパッケージ化および配布** チームへ

以下の場合も使用します：

- **ユーザーがスキルを作成/共有したいと言う場合**

このスキルは次の用途に最適です：

- チームワークフローの一部としてスキルを作成
- スキル作成+チーム通知が必要な内部ツールの構築
- スキル開発パイプラインの自動化
- チーム通知によるコラボレーティブスキル作成

## 主要機能

### 1. スキル作成

- SKILL.mdを備えた適切に構造化されたスキルディレクトリを作成
- 標準化されたscripts/、references/、assets/ディレクトリを自動生成
- 必要なメタデータを備えたYAML冒頭を自動生成
- 命名規則（ハイフンケース）を強制

### 2. スキル検証

- SKILL.mdフォーマットと必須フィールドを検証
- 命名規則をチェック
- パッケージ化前にメタデータの完全性を確保

### 3. スキルパッケージング

- 配布可能なzipファイルを作成
- すべてのスキルアセットとドキュメントを含む
- パッケージング前に自動的に検証を実行

### 4. Rue経由のSlack統合

- 作成されたスキル情報を指定されたSlackチャネルに自動的に送信
- スキルメタデータ（名前、説明、リンク）を共有
- チーム発見のためにスキル概要を投稿
- スキルファイルへの直接リンクを提供

## 仕組み

スキル作成と検証ワークフローを実行し、その後Rueを使用してSlackチャネルに自動的に通知を送信します。
