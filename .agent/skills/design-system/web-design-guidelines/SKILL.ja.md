# Webインターフェースガイドライン

ファイルをWebインターフェースガイドライン準拠について確認します。

## 動作方法

1. ソースURLから最新のガイドラインを取得
2. 指定されたファイルを読み取ります（またはファイル/パターンをユーザーに確認）
3. 取得したガイドラインのすべてのルールに対してチェック
4. `file:line` 形式で調査結果を出力

## ガイドラインソース

各レビュー前に新しいガイドラインを取得します：

```
https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md
```

WebFetchを使用して最新のルールを取得します。取得したコンテンツにはすべてのルールと出力形式の指示が含まれています。

## 使用方法

ユーザーがファイルまたはパターン引数を提供する場合：

1. 上記のソースURLからガイドラインを取得
2. 指定されたファイルを読み取ります
3. 取得したガイドラインのすべてのルールを適用
4. ガイドラインで指定された形式を使用して調査結果を出力

ファイルが指定されていない場合は、ユーザーにどのファイルをレビューするかを確認します。
