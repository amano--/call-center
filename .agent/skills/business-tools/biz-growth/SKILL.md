# Business Growth Skills

このスキルは「SNS・ビジネス・資料作成・アフィリエイト・プロンプト設計」を最短で成果につなげるための制約とルールを定義します。

## How to use

- `/biz-growth`
  Apply these constraints to any business/content creation work in this conversation.

- `/biz-growth <content>`
  Review the content against all constraints below.

## Mindset (思考基準)

- **Accuracy > Emotion** (精度 > 感情)
- **Reproducibility > Vibe** (再現性 > ノリ)
- **Logic > Assumption** (論理 > 思い込み)

## Output Rules (出力ルール)

- MUST start with the **Conclusion**.
- MUST follow with **Evidence/Rationale**.
- MUST end with **Actionable Proposals**.
- MUST limit key points to a maximum of **3**.
- NEVER use redundant paraphrasing.

## Quality Standards

- MUST assume commercial quality for all deliverables.
- SHOULD consider these elements for visual/creative descriptions:
  - `high detail`, `cinematic lighting`, `high contrast`, `clear composition`, `commercial use assumption`
- MUST explicitly describe Camera Work, Lighting, and Composition when proposing video/visual structures.

## Research Tasks

- MUST prioritize primary sources or official public data for numbers.
- NEVER guess uncertain information.
- MUST create a **Comparison Table**.
- MUST list **3-5 similar cases/competitors**.
- MUST summarize with "Conclusion and Implications".

## Documentation / Slides

- Structure: **Conclusion** -> **3 Key Points** -> **Draft Structure**.
- Rule: **1 Slide = 1 Message**.
- MUST simplify technical terms.
- MUST include **1 Analogy/Example**.

## Product / Service Introduction

- Structure:

  1. **Benefit**
  2. **Reader's Pain**
  3. **Solution Image**
  4. **One-liner Catchphrase**

- NEVER use exaggerated claims.
- SHOULD convert definitive assertions into impression-based expressions (avoid legal risks).
- For X (Twitter): Target **180-200 characters**, designed for thread expansion.
